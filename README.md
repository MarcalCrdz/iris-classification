# 🌸 Iris Classification - Machine Learning

Projeto de Machine Learning para classificação de espécies de flores utilizando o dataset Iris.

## 🚀 Objetivo
Desenvolver um modelo capaz de identificar a espécie de uma flor com base em suas características físicas.

## 📊 Dataset
Utilizado o dataset clássico Iris, contendo informações sobre:
- Comprimento da sépala
- Largura da sépala
- Comprimento da pétala
- Largura da pétala

Espécies:
- Setosa
- Versicolor
- Virginica

## 🧠 Tecnologias utilizadas
- Python
- Pandas
- Seaborn / Matplotlib
- Scikit-learn

## ⚙️ Etapas do projeto
- Carregamento e tratamento dos dados
- Análise exploratória (EDA)
- Visualização com gráficos (pairplot e correlação)
- Separação treino/teste
- Treinamento com Random Forest
- Avaliação com acurácia e matriz de confusão

## 📈 Resultados
O modelo atingiu aproximadamente **100% de acurácia** no dataset de teste.

## 💻 Mini sistema
Foi desenvolvido um sistema interativo em Python que permite inserir os dados da flor e receber a previsão da espécie em tempo real.
